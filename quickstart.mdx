---
title: Quickstart
description: Get up and running with the CS Interview Prep Platform in under 10 minutes
---

Get the CS Interview Prep Platform running locally and start your first interview session.

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/pranav9989/be.git
    cd be
    ```
  </Step>

  <Step title="Backend Setup">
    Create and activate a virtual environment:

    <CodeGroup>
    ```bash Windows
    python -m venv myenv
    myenv\Scripts\activate
    ```

    ```bash macOS/Linux
    python3 -m venv myenv
    source myenv/bin/activate
    ```
    </CodeGroup>

    Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
  </Step>

  <Step title="Environment Configuration">
    Create a `.env` file in the root directory:

    ```bash .env
    MISTRAL_API_KEY=your_mistral_api_key_here
    SECRET_KEY=your-secret-key-here
    JWT_SECRET_KEY=your-jwt-secret-here
    ```

    <Note>
    Get your Mistral API key from [console.mistral.ai](https://console.mistral.ai/)
    </Note>
  </Step>

  <Step title="Build Knowledge Base">
    Prepare and index the knowledge base:

    ```bash
    python scripts/prepare_kb.py
    python scripts/reindex_mistral.py
    ```

    This creates the FAISS vector index for DBMS, OOP, and OS topics.
  </Step>

  <Step title="Frontend Setup">
    In a new terminal, navigate to the frontend directory:

    ```bash
    cd frontend
    npm install
    ```
  </Step>

  <Step title="Start the Application">
    **Terminal 1 - Backend:**
    ```bash
    python backend/app.py
    ```

    **Terminal 2 - Frontend:**
    ```bash
    cd frontend
    npm start
    ```

    The app will open at `http://localhost:3000`
  </Step>

  <Step title="Create Your Account">
    1. Click **Sign Up**
    2. Fill in your details (username, email, password, full name)
    3. Log in with your credentials
  </Step>

  <Step title="Start Your First Session">
    Choose from:
    - **Mock Interview**: Resume-based personalized questions
    - **Technical Q&A**: Practice DBMS, OOP, or OS concepts  
    - **Coding Practice**: SQL and Pandas problem solving
  </Step>
</Steps>
